# Backend Monalisa
API em Java com Spring Boot que representa o banco digital "Monalisa"

<hr>

<h2>Rota para acessar o Swagger-UI:</h2>

- http://contexto/swagger-ui/index.html

<hr>

<h2>Endoints da API:</h2> 
<ul>
<li>Usuario<br>
  - Get<br>
  - Put<br>
  - Post<br>
  - Delete</li>
<li>Fatura<br>
  - Get<br>
  - Put<br>
  - Gete</li>
<li>Compra<br>
  - Post<br>
  - Get</li>
<li>Health<br>
  - Get</li>
<li>Login<br>
  - Post</li>
<li>Conta<br>
  - Get</li>
</ul>

<hr>
