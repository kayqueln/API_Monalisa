# Backend Monalisa
API em Java com Spring Boot de um banco digital chamado "Monalisa"

<hr>

<h2>Endoints da API:</h2> 
- Usuario
  - Get
  - Put
  - Post
  - Delete
- Fatura
  - Get
  - Put
  - Get
- Compra
  - Post
  - Get
- Health
  - Get
- Login
  - Post
- Conta
  - Get

<hr>

<h2>Tecnologias usadas:</h2>
- Java
- Spring Boot
- JPA
- SQL
- H2 (no ambiente de Dev)
- Spring Security
- JWT
